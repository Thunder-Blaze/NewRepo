## 1. Create a Fork of this Repo

## 2. Cloning
```bash
git clone <Fork_Url>
```

## 3. Branching
```bash
git branch <NewBranchName>
git checkout <NewBranchName>
            ||
git checkout -b <NewBranchNow>
```

## 4. Add your files
```bash
git add .
```

## 5. Commit your Changes
```bash
git commit -m "Commit Message"
```

## 6. Push your changes
```bash
git push -u origin <NewBranchName>
```
